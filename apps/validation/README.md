# app-validation

This private package is used to validate the gltf asset pipeline.

| Key       | Action                      |
| --------- | --------------------------- |
| `{space}` | stop animation              |
| `c`       | use cube texture            |
| `h`       | use hdr texture             |
| `l`       | toggle ambient light        |
| `e`       | toggle environment lighting |
| `b`       | toggle background           |
| `f`       | toggle blur                 |
| `r`       | Reset scene                 |

See keyboard commands in source.
